# Video Subtitle Search

## Build & Run

- Install Project

```bash
$ git clone https://github.com/abughalib/subtitle_search
```

- Change Directory

```bash
$ cd subtitle_search
```

- Install Requirements

```bash
$ pip install -r requirements.txt
```

- Migrate Database

```bash
$ python manage.py migrate
```

- Run Project

```bash
$ python manage.py runserver
```

- Open Browser

```bash
http://localhost:8000/
```
